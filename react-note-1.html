React Note 1

创建一个空的React项目

安装npm

安装node

brew install node

查看npm版本

npm -v

创建项目

脚手架工具create-react-app

npm install -g create-react-app

tips: 加速下载速度

npm config set registry https://registry.npm.taobao.org
npm config get registry

创建项目

create-react-app react_app
cd react_app
npm start

在项目下可以看到package.json文件如下

{
  "name": "react_app",
  "version": "0.1.0",
  "private": true,
  "dependencies": {
    "@testing-library/jest-dom": "^4.2.4",
    "@testing-library/react": "^9.5.0",
    "@testing-library/user-event": "^7.2.1",
    "react": "^16.13.1",
    "react-dom": "^16.13.1",
    "react-scripts": "3.4.1"
  },
  "scripts": {
    "start": "react-scripts start",
    "build": "react-scripts build",
    "test": "react-scripts test",
    "eject": "react-scripts eject"
  },
  "eslintConfig": {
    "extends": "react-app"
  },
  "browserslist": {
    "production": [
      ">0.2%",
      "not dead",
      "not op_mini all"
    ],
    "development": [
      "last 1 chrome version",
      "last 1 firefox version",
      "last 1 safari version"
    ]
  }
}

开头的name为工程名，version表示版本号，dependencies表示工程依赖，script是打包运行的脚本

作为一个java后端，有一种maven GAV坐标既视感

工程结构分析

node_modules是工程依赖的包

public/index.html是单页面应用SPA(single page application) 截取body的部分如下

<body>
    <noscript>You need to enable JavaScript to run this app.</noscript>
    <div id="root"></div>
  </body>

src/index.js是整个应用的入口js代码

import React from 'react';
import ReactDOM from 'react-dom';
import './index.css';
import App from './App';
import * as serviceWorker from './serviceWorker';

ReactDOM.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
  document.getElementById('root')
);

src/App.js表示App组件 注意所有的class使用className代替

import React from 'react';
import logo from './logo.svg';
import './App.css';

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <img src={logo} className="App-logo" alt="logo" />
        <p>
          Edit <code>src/App.js</code> and save to reload.
        </p>
        <a
          className="App-link"
          href="https://reactjs.org"
          target="_blank"
          rel="noopener noreferrer"
        >
          Learn React
        </a>
      </header>
    </div>
  );
}

export default App;
