React Note 1

Date 六 06 六月 2020 By Hester Category Note.

创建一个空的React项目

安装npm

安装node

brew install node

查看npm版本

npm -v

创建项目

脚手架工具create-react-app

npm install -g create-react-app

tips: 加速下载速度

npm config set registry https://registry.npm.taobao.org
npm config get registry

创建项目

create-react-app react_app
cd react_app
npm start
